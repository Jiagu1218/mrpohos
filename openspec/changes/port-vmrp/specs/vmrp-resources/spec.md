## 新增需求

### 需求:资源文件打包
系统必须将 MRP 模拟器运行所需的资源文件打包到 HarmonyOS 应用中。

#### 场景:打包完整性
- **当** 应用构建完成
- **那么** rawfile 目录中必须包含 cfunction.ext、mythroad 目录结构（含字体文件）、启动器 mrp

### 需求:首次启动资源释放
系统必须在首次启动时将 rawfile 中的资源文件复制到 HarmonyOS 沙箱目录。

#### 场景:首次启动
- **当** 应用首次启动且沙箱目录中不存在 mythroad 文件夹
- **那么** 必须将 rawfile 中的所有资源文件复制到沙箱目录的对应位置

#### 场景:非首次启动
- **当** 应用启动且沙箱目录中已存在资源文件
- **那么** 禁止重复复制，直接使用现有资源

### 需求:资源文件路径约定
资源文件在沙箱中的目录结构必须与 MRP 手机上的目录结构一致。

#### 场景:目录结构
- **当** 资源释放完成
- **那么** 沙箱根目录下必须包含 `cfunction.ext`、`mythroad/system/gb12.uc2`、`mythroad/system/gb16.uc2`、`mythroad/dsm_gm.mrp` 等文件
